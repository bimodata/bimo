# Charger des données

Comme évoqué dans l'[introduction](../../intro/intro.mdx), l'objectif de Bimo est de faciliter le traitement de données de toutes sources grâce à des plugins. Les plugins ne sont toutefois pas encore disponibles en Open Source. (N'hésitez pas à nous contacter pour évoquer vos besoins - cela pourrait nous motiver à accélérer la publication d'un plugin particulier.)

Avec les modules publiés pour le moment, il est possible de charger facilement des données qui auraient été exportées depuis HASTUS avec un OIG standard. Voir [charger des données depuis Hastus](./from-hastus.mdx).

Une autre option est de mettre vos données au format JSON, avec une structure correspondant au modèle-objet Bimo. Voir [charger des données JSON](./from-json.mdx).

import DocCardList from '@theme/DocCardList';
